# 📌 Project Overview

The **LinkedIn AI Profile Analyzer** is an intelligent system built using **Google Gemini AI** that analyzes a candidate’s LinkedIn profile and generates a **professional improvement report in PDF format**.

This project helps users improve their:
- 👤 Personal branding
- 🔍 Recruiter visibility
- 💼 Career opportunities
- 📊 Profile optimization

The system evaluates multiple sections of a LinkedIn profile and provides **AI-powered suggestions for improvement**.

---

# 🎯 Objective

The main objective of this project is to:

- Analyze LinkedIn profile content using AI
- Identify strengths and weaknesses
- Suggest improvements for better visibility
- Generate a structured professional report
- Improve resume and LinkedIn alignment

---

# ✨ Features

## 🧠 AI Analysis Features
- 📌 Headline Quality Analysis  
- 📖 About Section Evaluation  
- 🛠️ Skills Assessment  
- 💼 Experience Review  
- 🔑 Keyword Optimization for Recruiters  
- 📊 Resume Alignment Check  
- 🏆 Overall Profile Score (0–100)

---

## 📄 Output Features
- Automatically generates a **professional PDF report**
- Structured AI feedback
- Actionable improvement suggestions
- Clean and readable formatting

---

## ⚙️ System Features
- Input-based interactive system
- Uses Google Gemini API for intelligent analysis
- Stores output in organized folder structure
- Easy to run in local environment
